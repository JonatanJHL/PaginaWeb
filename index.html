class APIClient {
    constructor(baseURL) {
        this.baseURL = baseURL;
    }

    async fetchData(endpoint) {
        try {
            const response = await fetch(`${this.baseURL}${endpoint}`);
            if (!response.ok) throw new Error(`Error: ${response.statusText}`);
            return await response.json();
        } catch (error) {
            console.error('Error fetching data:', error);
        }
    }
}

Sistema de Gestión de Vacaciones

Este proyecto implementa un sistema robusto de gestión de vacaciones desarrollado con **HTML**, **PHP**, y **MySQL**, con integración de Google OAuth para autenticación segura. Los usuarios pueden gestionar sus solicitudes de vacaciones y obtener información relevante directamente desde un panel intuitivo. A continuación, se describen las principales funcionalidades:

Características del Sistema:

  • Autenticación: Inicio de sesión mediante Google OAuth para garantizar la seguridad y simplificar el acceso.
  • Panel de usuario:
    • Días de vacaciones usados y restantes.
    • Fecha límite para usar los días disponibles.
    • Visualización de solicitudes previas con estatus: **aprobadas**, **rechazadas**, o **en espera**.
  • Solicitud de vacaciones:
    • Selección de días directamente en un calendario interactivo.
    • Ordenación automática de días seleccionados (de menor a mayor).
    • Notificación automática al líder mediante **PHPMailer** al completar la solicitud.
  • Panel de líder:
    • Visualización de todas las solicitudes asignadas.
    • Opciones para aprobar o rechazar solicitudes.
    • Notificación automática al colaborador sobre el estado de su solicitud mediante correo electrónico.

Reconocimiento Facial con OpenCV

En este proyecto utilicé **OpenCV** para implementar un sistema de reconocimiento facial simple. El sistema es capaz de detectar y reconocer rostros, ditingiendo algunas emociones, utilizando modelos preentrenados.

Características:

  • Detección de rostros en imágenes y videos.
  • Identificación emociones en una persona.
  • Alertas en caso de intentos de acceso no autorizado.

Ejemplo de Código:

import cv2
from deepface import DeepFace

def detectar_emocion_en_imagen(ruta_imagen):
    # Cargar el modelo de detección de rostros de OpenCV
    face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + 'haarcascade_frontalface_default.xml')

    # Leer la imagen
    imagen = cv2.imread(ruta_imagen)
    if imagen is None:
        print("Error: No se pudo cargar la imagen.")
        return

    # Convertir a escala de grises para detección de rostros
    gray = cv2.cvtColor(imagen, cv2.COLOR_BGR2GRAY)
    rostros = face_cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5, minSize=(30, 30))

    if len(rostros) == 0:
        print("No se detectaron rostros en la imagen.")
        return

    for (x, y, w, h) in rostros:
        # Dibujar un rectángulo alrededor del rostro
        cv2.rectangle(imagen, (x, y), (x + w, y + h), (255, 0, 0), 2)

        # Recortar la región del rostro
        rostro = imagen[y:y + h, x:x + w]

        try:
            # Analizar la emoción usando DeepFace
            analisis = DeepFace.analyze(rostro, actions=['emotion'], enforce_detection=False)

            # Filtrar emociones solo entre "enojado", "triste" y "feliz"
            emociones_validas = {key: analisis['emotion'][key] for key in ['angry', 'sad', 'happy']}
            emocion_predominante = max(emociones_validas, key=emociones_validas.get)

            # Mostrar la emoción detectada
            cv2.putText(imagen, emocion_predominante, (x, y - 10), cv2.FONT_HERSHEY_SIMPLEX, 0.9, (255, 0, 0), 2)
        except Exception as e:
            print(f"Error al analizar emociones: {e}")

    # Mostrar la imagen con el rostro detectado y la emoción
    cv2.imshow('Detección de Emociones', imagen)
    cv2.waitKey(0)
    cv2.destroyAllWindows()

# Ruta de la imagen
ruta_imagen = "ruta_a_tu_imagen.jpg"

# Detectar emoción en la imagen
detectar_emocion_en_imagen(ruta_imagen)
