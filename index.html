Proyectos destacados

En producción

AI-Powered Data Assistant (CAPA 8)

Sistema interno de consulta de datos empresariales en lenguaje natural, deployado en producción en SERVNET. Integra DeepSeek vía OpenRouter para traducir preguntas coloquiales a SQL dinámico sobre la base de datos operativa. Incluye RBAC embebido en el prompt, memoria conversacional multi-turno y caché de consultas frecuentes por hash para reducir latencia y costo de tokens. Permite que usuarios no técnicos obtengan reportes sin intervención del equipo de desarrollo.

PHP OpenRouter DeepSeek MySQL LLM RBAC NLP

Task Manager Pro

App full-stack de gestión de proyectos con tablero Kanban, drag & drop, comentarios, archivos adjuntos y analytics de productividad. Backend Node.js + Express con JWT (refresh token rotation) y notificaciones por email vía Resend. Frontend React + Vite con Chart.js. Containerizado con Docker.

Node.js Express React MySQL JWT Docker Chart.js

CFDI 4.0 — Validación Fiscal

Módulo de autoservicio fiscal para que clientes de SERVNET actualicen su información de facturación cumpliendo con los requisitos SAT. Flujo completo: captura → validación administrativa → sincronización automática con WHMCS vía API. Eliminó la intervención manual del equipo de soporte en actualizaciones fiscales.

PHP WHMCS API MySQL PHPMailer
En producción

PDF Parser — Constancia Fiscal

Automatización de extracción de datos fiscales desde PDFs del SAT en el browser. Lee el archivo con PDF.js, extrae RFC, razón social, régimen y dirección con regex calibradas al formato oficial, y pre-llena automáticamente el formulario de registro. Elimina captura manual y errores de tipeo en el onboarding de clientes.

JavaScript PDF.js RegEx HTML5
En producción

Sistema de Gestión de Colaboradores

Plataforma interna de RRHH construida desde cero con PHP y MySQL. Autenticación Google OAuth 2.0, control de vacaciones y ausencias con FullCalendar (flujo de aprobación), organigramas dinámicos con GoJS y módulo de asignación de equipos. Consolidó procesos que vivían en hojas de cálculo y emails.

PHP OAuth 2.0 FullCalendar GoJS MySQL
En producción

Dashboard de Facturación con Aging

Dashboard de cartera vencida para el área de cobranza. Consolida facturas en tiempo real, clasifica por antigüedad (0–30, 31–60, 61–90, 91–120, 121+ días) y calcula exposición en MXN y USD. Exportación a Excel lista para dirección. Reemplazó consolidación manual que tomaba horas al mes.

PHP MySQL Chart.js Excel
En producción

Integración Omnicanal WSAS–Slack

Integración bidireccional entre WSAS y Slack para unificar atención a clientes. Tickets y mensajes sincronizados en tiempo real vía webhooks; respuestas desde Slack reflejadas en WSAS vía API. El equipo de soporte opera desde Slack sin perder trazabilidad en la plataforma oficial.

PHP Slack API Webhooks WHMCS

SQLQuest — SQL Learning RPG

Juego educativo browser-based donde avanzas por capítulos resolviendo retos con SQL real. Motor SQL interpretado en JavaScript puro, sin backend. Base de datos ficticia con relaciones realistas (héroes, misiones, items, reinos). Integración con OpenRouter planeada para narrativa generativa con IA.
